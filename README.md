## 🚀 Overview
This project detects whether a message is Spam or Not Spam using Machine Learning (Naive Bayes + TF-IDF).

## 🧠 Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas

## ⚙️ How it works
1. Text is converted into numerical features using TF-IDF
2. Model predicts spam or not spam
3. Streamlit UI shows result
